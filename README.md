#  TaskHero: Advanced Task Management System
TaskHero is a robust, web-based productivity tool designed to help users manage daily tasks with integrated reminders, notifications, and an interactive puzzle feature for engagement.

##  Key Features
- **Full CRUD Functionality:** Create, track, complete, and delete tasks seamlessly.
- **Smart Notifications:** Integrated scripts (`get_notifications.php`, `Tasks/Reminders.php`) to keep users on track.
- **Secure Authentication:** User registration and login system with session management.
- **Gamified Element:** Includes a `puzzle.js` component to provide a unique user experience.
- **Clean UI:** Responsive design powered by custom CSS.

##  Technical Stack
- **Backend:** PHP
- **Database:** MySQL
- **Frontend:** HTML5, CSS3, JavaScript (AJAX for notifications)
- **Server:** XAMPP / Apache

##  Setup Instructions
1. Clone this repository to your local `htdocs` folder.
2. Open **phpMyAdmin** and create a database named `taskhero`.
3. Import the `taskhero.sql` file provided in the repository.
4. Configure your database credentials in `db.php`.
5. Access the application via `localhost/task_manager_TaskHero_with_puzzle/`.
